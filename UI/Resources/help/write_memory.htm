[Write Memory] Overview

 

Flash Tool provides Write Memory function for users to write specified file to specified address on the flash.

 

·  Enter Write Memory Page

 

Write Memory Page can be shown by pressing “Window->Write Memory” on advanced mode.

 

 

·  Write Memory Steps

 

1.       Power off target

A handshake must be performed between Flash Tool and the target mobile phone, thus the target must initially be powered off, and connected to the PC via a download cable.

For USB download, please pull out both Cable and Battery.

 

2.       Select flash xml file

Please select the correct flash xml file according to the chip which is supposed to be executed. The flash xml file could be loaded by “choose” button by the side of "Download-XML" label or select in history list, and the file can be found in the load folder.

Please make sure the flash xml file has been assigned.

 

 

4.       Select file to write

Switch to write memory page, click the button with folder icon, select specific file to write.

 

 

5.       Configure begin address

Configure specific address from where the file will be written.

 

 

6.       Configure Region

Configure region from where the file will be written.

For UFS, UFS-LUA0, UFS-LUA1 and UFS-LUA2 are provided.

For EMMC, EMMC-BOOT1, EMMC-BOOT2 and EMMC-USER are provided.

For NAND, NAND-WHOLE are provided.

 

 

7.       Start write memory

Press “Write Memory” button to start downloading image.

 

8.       Power on target

Power on the target and the download procedure will begin.

 

9.       Stop write memory

Note that to interrupt the download process at any time, press the STOP button or F10.
