[Memory Test] Overview

 

For memory validation and testing, Flash Tool offers a Memory Test for users to test and verify the external memory devices RAM, eMMC and NAND flash.  In the test mode, the DA is downloaded into the target’s internal SRAM to check the handset’s memory devices. Flash Tool will parse the memory deceive configuration information from the Pre-Loader.

 

·  Memory Test Options

Flash Tool provides DRAM Flip Test method for users, and Auto and Manual can be selected.

 

1.       Auto

If “Auto” is selected, the memory test range is according to the target flash information reported by DA; No further information is required from user input.

 

  

 

2.       Manual

By selecting “Manual” option, the memory test range is according to the specified range. 

 

 

·  Memory Test steps

 

1.       Switch to Memory Test page

 

To enter the Test Mode, select the “MemoryTest” page.

 

 

2.     Select Memory Test Methods

 

You can configure the DRAM Flip Test option as mentioned below.

 

3.     Power off target

 

A handshake must be performed between Flash Tool and the target mobile phone, thus the target must initially be powered off, and connected to the PC via a download cable.

For USB download, please pull out both Cable and Battery.

 

4.       Select flash xml file

 

Please select the correct flash xml file according to the chip which is supposed to be executed. The flash xml file could be loaded by “choose” button by the side of "Download-XML" label or select in history list, and the file can be found in the load folder.

Please make sure the flash xml file has been assigned.

 

 

5.       Configure memory test options

 

Configure the memory test options mentioned above and select the expected power supply and Auto Reboot option on the BROM Connection setting.

 

 

 

6.       Start Memory Test

 

Press “Start” button in Memory Test page to start memory test procedure.

 

 

7.       Power on target

 

Power on the target and the memory test procedure will begin.

 

8.       Stop Memory Test

 

Note that to interrupt the memory test process at any time, press the STOP button or F10.
