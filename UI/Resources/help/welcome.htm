Welcome To Flash Tool

Thin-Modem Flash Tool is a cross platform application that could download built images and binary data to MediaTek smart phone chips. Now it supports Win 7 64bit and Ubuntu 10.4.

 

Format

This feature allows users to format phone UFS, EMMC or SDMMC flash.

 

Download

This feature is used to download built image or binary data to target.

 

 

 

 

Readback

This feature allows customer to read a length of

Memory from phone.

 

Memory Test

Thin-Modem Flash tool offers this feature for Users to test and verify the external memory Device RAM.

 

 

 

 

Brom Adapter

This feature allows user to send a executable program to RAM to run.

 

Write Memory

This feature allows user to write specified file to specified address on the flash.

 

 

 

 

DL Cert

Used to download certification file to flash .

 

Option

Thin-Modem Flash Tool also provides option function for user to customize their tool and set some parameters for download, and so on.

 

 

 

 

Console

This feature allows customer to run Flash Tool in console mode.
