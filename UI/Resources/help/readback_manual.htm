Manual Readback

 

·  Manual Readback Options

 

1.       Switch to Manual Readback

Click "Manual" radio button to switch to Manual Readback operation .

 

 

2.       Assign Readback Items

Click the”Add” button to add a read back item, and “Remove” button will remove the selected item.

 

 

Or you can click the mouse right button to add or remove readback items.

 

 

2.       Edit Readback Items

·  Select Readback Item

You can select the readback item which you want to do readbak by check the checkbox.

And you can select or dis-select all the readback items by check the title checkbox,

 

 

·  Edit Start Address

Double click the "Start Address" column cell to edit it..

 

 

·  Edit Length

Double click the "Length" column cell to edit it.

 

 

·  Edit Region

Double click the "Region" column cell to show the region list.

For UFS, UFS-LUA0, UFS-LUA1 and UFS-LUA2 are provided.

For EMMC, EMMC-BOOT1, EMMC-BOOT2 and EMMC-USER are provided.

For NAND, NAND-WHOLE are provided.

 

 

·  Edit File

Double click the "File" column cell to select expected file path and file name.

 

 

·  Manual Readback Steps

 

·  Power off target

A handshake must be performed between Flash Tool and the target mobile phone, thus the target must initially be powered off, and connected to the PC via a download cable.

For USB download, please pull out both Cable and Battery.

 

·  Select flash xml file

Please select the correct flash xml file according to the chip which is supposed to be executed. The flash xml file could be loaded by “choose” button by the side of "Download-XML" label or select in history list, and the file can be found in the load folder.

Please make sure the flash xml file has been assigned.

 

 

·  Config Manual Readback Options

Config manual reaback options mentioned above .

 

·  Start Manual Readback

Press “Read Back” button to start readback procedure.

 

 

·  Power on target

Power on the target and the readback procedure will begin.

 

·  Stop Readback

Note that to interrupt the readback process at any time, press the STOP button or F10.
