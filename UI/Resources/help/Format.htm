[Format] Overview

 

·  Format Options

Flash Tool provides Auto Format and Manual Format for users.

 

1.       Auto Format

If “Auto Format Flash” is selected, the format range is according to the target flash information reported by DA; No further information is required from user input.

 

 

 

 

 

2.       Manual Format

By selecting “Manual Format Flash”, the flash is formatted according to the specified range. 

For UFS, UFS-LUA0, UFS-LUA1 and UFS-LUA2 are provided.

For EMMC, EMMC-BOOT1, EMMC-BOOT2 and EMMC-USER are provided.

For NAND, NAND-WHOLE and NAND-BMT are provided, and NAND-BMT is ONLY for Advanced Mode.

 

 

 

·  Format Steps

 

1.       Power off target

A handshake must be performed between Flash Tool and the target mobile phone, thus the target must initially be powered off, and connected to the PC via a download cable.

For USB download, please pull out both Cable and Battery.

 

2.       Select flash xml file

Please select the correct flash xml file according to the chip which is supposed to be executed. The flash xml file could be loaded by “choose” button by the side of "Download-XML" label or select in history list, and the file can be found in the load folder.

Please make sure the flash xml file has been assigned.

 

 

3.       Configure format options

Configure the format options instructed above and select the expected power supply and Auto Reboot option on the BROM Connection setting.

 

 

 

4.       Start Format

Press “Start” button in format page to start format operation.

 

 

5.       Power on target

Power on the target and the format procedure will begin.

 

6.       Stop Format

Note that to interrupt the format process at any time, press the STOP button or F10.
