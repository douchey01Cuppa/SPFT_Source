[DL Cert] Overview

 

·  DL Cert Steps

 

1.       Power off target

A handshake must be performed between Thin-Modem Flash Tool and the target mobile phone, thus the target must initially be powered off, and connected to the PC via a download cable.

For USB download, please pull out both Cable and Battery.

 

2.       Select flash xml file

Please select the correct flash xml file according to the chip which is supposed to be executed. The flash xml file could be loaded by “choose” button by the side of "Download-XML" label or select in history list, and the file can be found in the load folder.

Please make sure the flash xml file has been assigned.

 

 

3.       Select certification file

Click button on the right to select the certification file.

 

 

4.       Start Download Cert

Press "DL Cert " button to start download Cert procedure.

 

 

5.       Power on target

Power on the target and the format procedure will begin.

 

6.       Stop

Note that to interrupt the format process at any time, press the STOP button or F10.
