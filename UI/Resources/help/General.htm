General Option

 

General Setting provides personalized customization for language and log handling.

 

·  General

1. Whether show welcome page when tool startup can be configured by changing state of “General” -> “Show Welcome page when startup”.

 

 

2. Whether need to check device storage life cycle check after device connection can be configured by changing state of “General” -> “Storage Life Cycle Check ”.

 

 

·  Language

Language can be configured in “General”->”Language”.

Up to now, English, Simplified Chinese and Traditional Chinese are available.

 

 

·  Tracing

Configure options relative with logs.

·  Enable/Disable Log

Press Ctrl + Alt +M to enter or exit “Runtime Tracing Mode” or change the state of “General”->”Tracing”->” Enabling tracing ”.

 

 

·  Configure log information

1.       Log File Path

Press the "..." button to select log path .

 

 

 

You can select a new log file path to store the log files generated next time the Flash Tool starts.

 

 

After log file path selected, the selected log path will be shown on the tracing page, and the log path will take effect next time the Flash Tool starts.

 

 

2.       Log Files Clean

The Flash Tool offers four period: One Day, One Week, One Month and Three Month. The log files before the selected period will be clean automatically.

 

3.       DA Log Level

The Flash Tool offers da log level setting section, used to configure tracing log level. DEBUG, INFO, WARN and ERROR are available.

 

 

4.       DA Log Channel

The Flash Tool offers da log channel setting section, used to configure tracing log channel. USB and UART are available.
